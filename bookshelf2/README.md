## как запустить

из корня проекта:
1. собрать проект `swift build -v`
2. запустить проект `swift run`

## поддерживаемые команды/сценарии
1. добавление книги
    - введите цифру 1
    - введите данные книги по условию
2. удаление книги
    - введите цифру 2
    - введите id книги
3. вывод списка книг
    - введите цифру 3
4. фильтрация по автору, названию, году, тегам
    - введите цифру 4
    - введите номер значения фильтрации из предложенного списка
>Варианты фильтрации:
>1.) Название
>2.) Автор
>3.) Жанр
>4.) Тег
>5.) Год
